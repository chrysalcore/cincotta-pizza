## Description

**Cincotta Pizza Landing Page** was created to provide a landing for the fictional restaurant with the same name. They offer a wide variety of flavors in a fresh and healthy Italian style.

The site features a single page with several sections. Upon entering, you'll notice pleasant green colors of various shades, of course, with a strong contrast to the red, designed to direct visitors to that long-awaited reservation. Perfect for daytime outdoor restaurants, with a healthy and friendly vibe.

The code was written using React, state management, componentization, and the BEM methodology, adhering to best practices, responsiveness, accessibility, and other web standards.

## Main Features

- Advanced state management.
- Components with JavaScript.
- Responsive UI/UX: Interface designed with CSS that adapts to all devices.
- Smooth scrolling and a fluid user experience.

## Local Installation and Use

Follow these steps to run the project on your local machine.

### Prerequisites

- [Node.js](https://nodejs.org/es/)
- [NPM](https://www.npmjs.com)

### Steps

1. Clone the repository

    ```bash
    git clone https://github.com/chrysalcore/cincotta-pizza.git
    cd cincotta-pizza
    ```

2. Install dependencies

    ```bash
    npm install
    ```

3. Run development mode

    ```bash
    npm run dev
    ```
